What I've been building.

Autonomous agent · 2026

Horizon

Horizon is my always-on agent. I send it a task in Telegram, in Russian or English. It plans the work, hands it to the right specialist mini-agent, and reports back when the work is done. It lives on a small Linux server and keeps working while I sleep.

Before anything runs, a council of AI reviewers checks the plan and votes.

Python · Claude CLI · Telegram Bot API · Linux VPS

Voice assistant · prototype

Jarvis

Jarvis is a voice assistant prototype for my Mac. It wakes on "Hey Jarvis", with wake-word detection running locally, and holds a conversation in Russian or English. Voice needs an answer fast, but real work takes minutes: a fast voice model handles the talking, real tasks go to a Claude agent, and the session stays open until the answer comes back.

openWakeWord · OpenAI Realtime API · Claude Agent SDK · MCP

Study tool · web app

Case Dojo

Case Dojo is the app I built to study business thinking. Flashcards with spaced repetition, quizzes, worked cases, and a teach-back mode where I explain a framework in my own words and grade myself against a rubric. It runs offline with no account and no API key. Add a key, and an AI coach starts interviewing you on fresh cases; the key stays on a small server proxy and never reaches the browser.

Terms stay in English, explanations in Russian. That's how my head works anyway.

React · Vite · SM-2 spaced repetition · Express · Claude API

Personal tracker · web app

Summer Calendar

Summer Calendar is the tracker for my summer plan: a rotating day schedule, weekly checklists, streaks, and progress towards my goals. Data is plain JSON files in a git repo, so nothing depends on someone else's cloud. It starts at login and shows today's plan first thing in the morning.

Python · vanilla JS · JSON + git · launchd
