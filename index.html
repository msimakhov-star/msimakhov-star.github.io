Agent system

Horizon

An always-on agent I message in Telegram, in Russian or English. It plans the task, a council of AI reviewers votes on the plan, a specialist agent does the work, and it reports back from its own Linux server. It keeps working while I sleep.

Python · Claude CLI · Telegram Bot API · Linux VPS

Study tool

Case Dojo

A web app I built to study business thinking: spaced-repetition flashcards, worked cases, and a teach-back mode where I explain a framework in my own words and grade myself. It runs offline; add an API key and an AI coach starts interviewing you on fresh cases.

React · Vite · SM-2 · Express · Claude API

Personal tracker

Summer Calendar

The tracker for my summer plan: a rotating day schedule, weekly checklists, and streaks. Data is plain JSON files in a git repo, so nothing depends on someone else's cloud. It starts at login and shows today's plan first thing in the morning.

Python · vanilla JS · JSON + git · launchd
